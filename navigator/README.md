# ATT&CK Navigator layers — detection-diary

Auto-generated by `tools/generate_navigator.py`. **Do not edit by hand.**

## Files

| File | What it shows |
|---|---|
| `coverage-enterprise.json` | Cumulative Enterprise ATT&CK heatmap — every technique seen across all cases, scored by how many cases feature it. The hot cells are the diary's most-covered techniques. |
| `coverage-ics.json` | Same, for ICS ATT&CK (`T0xxx`) — present only if the diary has ICS cases. |
| `cases/<slug>.json` | One layer per case — just that case's techniques highlighted. |

## How to open

1. Go to the **[ATT&CK Navigator](https://mitre-attack.github.io/attack-navigator/)**.
2. **Open Existing Layer -> Upload from local**, pick a JSON file.
   (Or host the repo on GitHub Pages and paste the raw JSON URL.)
3. The cumulative layer uses a pale -> blue -> red gradient: redder = covered by
   more cases.

Score on the cumulative layer = number of cases featuring that technique, so the
heatmap doubles as a coverage-gap map: cold/empty cells are techniques the diary
has not yet taught.
